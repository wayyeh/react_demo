# 新一代javascript版本列表  
es6，也就是ECMAScript 2015，它是es5的升级版，增加了很多js新特性，这些新特性非常有帮助于提高我们的开发效率和质量。

es6于2015年6月份发布，其后的每一年都会提出一些新特性，于是就出现了es7、es8、es9...
## ECMAScript 2015(ES6)
* let 和 const
* class类
* 模块化(es-module)
* 箭头函数 (arrow function)
* 函数参数默认值
* 模板字符串
* 解构赋值
* 延展操作符 `...`
* Promise
